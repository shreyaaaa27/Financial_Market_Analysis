# Financial Market Pattern Recognition using Machine Learning

## Project Overview
This project implements a machine learning-based pattern recognition system for financial markets using real-time MetaTrader 5 data.

The system classifies market trends as:
- Bullish (price likely to go up)
- Bearish (price likely to go down)

## Models Used
- Logistic Regression
- Support Vector Machine (SVM)

## Features
- Returns
- 5-period Moving Average
- 10-period Moving Average

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report
- ROC Curve
- Wilcoxon Signed-Rank Test

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Start MetaTrader 5 and login

3. Run:
   python main.py
